OpenSCENARIO XSD Schema Files
===============================

These XSD schema files are part of the ASAM OpenSCENARIO standard.

Source: ASAM e.V. (Association for Standardization of Automation and Measuring Systems)
Website: https://www.asam.net/standards/detail/openscenario/

License: 
The OpenSCENARIO standard and its associated XSD files are copyrighted by ASAM e.V.
Usage is typically permitted for implementation purposes.

Please verify the current license terms at:
https://www.asam.net/standards/detail/openscenario/

Files needed in this directory:
- OpenSCENARIO.xsd (version 1.2)

To obtain the official XSD files:
1. Visit https://www.asam.net/standards/detail/openscenario/
2. Download the OpenSCENARIO 1.2 release
3. Extract OpenSCENARIO.xsd to this directory
